Deskripsi

Homerun is a fullscreen launcher with content organized in tabs. A tab is composed of several ‚Äúsources‚Äù. A source can provide one or more sections to a tab. Homerun comes with a few built-in sources, but custom sources can be written using libhomerun.

Instalasi

Berhubung saya menggunakan distribusi Arch Linux, saya mencari PKGBUILD untuk di-build di AUR. Hasil pencarian di AUR homerun ada 2 versi, yaitu versi git dan non-git. Ada 2 metode instalasi, yang pertama yang termudah dengan menggunakan yaourt, dan yang kedua metode yang sedikit ribet.

  • Instalasi dengan yaourt

  • Download; ekstrak; build

    1. Download kdeplasma-applets-homerun.tar.gz

    2. Ekstrak

    3. Build

      Setelah menjalankan perintah makepkg tinggal menunggu proses yang berjalan dan berikan konfirmasi instalasi saat ditanyakan.

Penampakan

  • Home

    Tab Home pada Homerun berisikan Favorite Applications (dapat ditambahkan), Favorite Places dan Recent Documents

  • Applications

    Tab Applications ini berisikan kategori aplikasi yang terinstall

  • Files

    Tab Files berisikan berkas-berkas yang ada $HOME

  • Power

    Tab Power berisikan shorcut untuk session dan power
