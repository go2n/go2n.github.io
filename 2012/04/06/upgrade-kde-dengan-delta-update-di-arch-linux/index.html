Upgrade KDE dengan Delta Update di Arch Linux

06 April 2012 | linux, KDE

Waah ternyata lumayan lama juga saya ndak nulis di blog ini. Kali ini saya akan berbagi tulisan mengenai pengalaman saya meng-upgrade Arch Linux dengan delta update. Pada awalnya saya sempat ragu menggunakan delta karena delta update sendiri belum didukung secara official. Selain itu mirror yang menyediakan paket delta baru 1 yaitu http://delta.archlinux.fr/. Namun pada akhirnya saya nekat saja, kalau ndak nekat kapan saya punya pengalaman upgrade menggunakan delta? üòÅ

Upgrade menggunakan delta memang bersahabat sekali dengan yang fakir bandwith. Saya sewaktu upgrade KDE dari 4.8.1 ke 4.8.2 total download paket bisa sampai ratusan megabyte namun setelah menggunakan delta update paket yang di-download cukup 51.53 MB!
