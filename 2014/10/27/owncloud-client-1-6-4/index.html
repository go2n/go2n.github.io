ownCloud client 1.6.4

27 Oktober 2014 | linux

Hae‚Ä¶ Tanggal 23 Oktober kemarin ownCloud client 1.6.4 dirilis loh, sudah update belum? Berikut ini changelog ownCloud client 1.6.4:

  • Fix startup logic, fixes bug #1989
  • Fix raise dialog on X11
  • Win32: fix overflow when computing the size of file > 4GiB
