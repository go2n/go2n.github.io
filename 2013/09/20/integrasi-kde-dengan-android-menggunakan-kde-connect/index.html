Integrasi KDE dengan Android menggunakan KDE Connect

20 September 2013 | linux

Instalasi

  • Arch Linux: kdeconnect atau kdeconnect-git (AUR)
  • openSUSE 12.3: kdeconnect-kde (1 Click Install)
  • Android: KDE Connect

Instalasi dari kode sumber

  • Clone: git clone git://anongit.kde.org/kdeconnect-kde
  • Install dependensi: kdelibs, qjson
  • Kompilasi:
    • cmake -DCMAKE_INSTALL_PREFIX=kde4-config ‚Äìprefix`
    • make
    • sudo make install atau su -c "make install"

Penampakan:

Diuji coba dengan KDE 4.11.1 di openSUSE 12.3 dan ponsel Android Samsung Galaxy Fit.
